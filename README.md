`psycopgr` is a Python wrapper of [pgRouting](http://pgrouting.org/) with one purpose:

**Computing routes on real map for humans.**

Tested with

- Python 3.6.5
- PostgreSQL 11.2
- PostGIS 2.5.2
- pgRouting 2.6.2
- osm2pgrouting 2.3.6

## Installation

```sh
pip install psycopgr
```

or

```sh
pipenv install psycopgr
```
